# lyrica
делаю штуку которая берет трек со спотифая, прогоняет его через нейронку и синхронизирует текст с музыкой, послогово, в такт. при этом оформление подстраивается под вайб: цвета из обложки, темп и атмосфера трека влияет на анимацию, жанр на верстку. так, текст у динамичного трека будет резко влетать крупными буквами, у ambient'а плавно таять.

стек: python (whisper, demucs, fastapi) + html/css/js на фронте. spotify api, genius, yt-dlp.

если интересно - велком, есть что поделать и на бэке и на фронте
